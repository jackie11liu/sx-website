    关于双欣

    内蒙古双欣能源化工有限公司（原名称为内蒙古双欣资源集团有限公司）成立于2000年7月，位于内蒙古鄂尔多斯市棋盘井工业园区，注册资本13.11亿元，法定代表人乔玉华，主要经营可降解高分子聚乙烯醇（PVA）、高强高模纤维、聚乙烯醇膜、聚乙烯醇缩丁醛（PVB）、煤基活性炭、电力、煤炭及煤基新材料的研发、生产、销售；液化天然气（LNG）、碳化钙的研发、销售；聚乙烯醇高分子材料产业链及其应用开发、研究。
